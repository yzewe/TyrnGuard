# TyrnGuard
**Advanced WireGuard over TURN Tunnel Client**

---

</div>

**TyrnGuard** — это мобильное приложение для ОС Android, являющееся развитием проекта WDTT. Инструмент предназначен для инкапсуляции WireGuard-трафика в DTLS-соединения, что позволяет маскировать VPN-подключения под легальный медиа-трафик (видеозвонки) через TURN-серверы VK.

## Ключевые изменения и функции

Проект включает в себя актуальный апстрим (v1.0.6) и ряд дополнительных возможностей для управления и кастомизации:

* **Менеджер серверов:** Поддержка работы с несколькими конфигурациями и возможность быстрого переключения между ними.
* **Импорт и экспорт:** Механизм переноса настроек серверов между устройствами.
* **Расширенная настройка сети:** Возможность ручной правки параметров DNS и MTU для оптимизации соединения.
* **Системная интеграция:** Добавлен виджет для рабочего стола и плитка в шторке быстрых настроек (Quick Settings Tile).
* **Диагностика:** Встроенная функция проверки доступности сервера прямо из интерфейса.
* **Обновленный интерфейс:**
    * Интерактивный дашборд с расширенной кастомизацией.
    * Переработанная система анимаций и виброотклик.
    * Система онбординга для первичной настройки приложения.
    * Уведомления с подробным статусом (сбор данных, этап подключения).
    * Оповещения при разрыве соединения.

## Техническое описание

Приложение позволяет обходить системы глубокого анализа пакетов (DPI) путем изменения структуры трафика:

1. Трафик WireGuard упаковывается в DTLS.
2. Соединение устанавливается с доверенным сервером VK TURN.
3. Конечный VPS остается скрытым для внешнего наблюдателя, так как весь видимый трафик идет на легальные IP-адреса сервисов ВК.

## Инструкция по установке

### 1. Подготовка VPS
Для работы требуется сервер на базе Linux. Настройка осуществляется через вкладку «Деплой» внутри приложения:
- Введите данные SSH (IP, порт, логин и пароль).
- Укажите пароль для шифрования туннеля.
- Приложение автоматически установит необходимые зависимости и откроет порты 56000 и 56001 (UDP).

### 2. Получение хеша
- Создайте звонок в VK.
- Скопируйте идентификатор из ссылки (код после последнего слэша).
- Допускается использование до трех хешей одновременно (через запятую).

### 3. Подключение
- Вставьте хеши и пароль туннеля во вкладке «Туннель».
- Установите количество потоков (рекомендуется от 12 до 24).
- Используйте центральную кнопку на главном экране для запуска.
